## 🌐 Overview

Welcome to the **Agentic AI Portfolio Nexus**. This project is a bespoke, highly interactive personal portfolio designed with a "Cybernetic Intelligence Bureau" / "Rogue AI Operator" aesthetic. It leverages modern web technologies to deliver a unique, immersive experience through neo-brutalism, glassmorphism, floating nodes, neon highlights, and terminal-style console effects.

## ✨ Features

- **Futuristic UI/UX**: Dark interfaces, glowing neon borders (`Agent Green`, `Cyber Blue`, `Hot Pink`), and scanning sweeping terminal effects.
- **Dynamic Interactions**: Highly interactive Framer Motion animations, magnetic floating skill badges, and 3D tilting project cards.
- **Thematic Sections**:
  - `[ HOME ]`: Holographic agent avatars and boot-up scramble text sequences.
  - `[ EXPERIENCE ]`: Brutalist mission logs with multi-color glowing fiber-optic scroll lines.
  - `[ PROJECTS ]`: Interlocking masonry grids showcasing deployed protocols.
  - `[ SKILLS ]`: Interactive, hovering magnetic nodes.
  - `[ ARCHIVES ]`: Ancient algorithms decrypted in dark amber and gold terminal UIs.
  - `[ CONTACT ]`: Secure encrypted transmission forms.
- **Fluid Navigation**: A responsive, animated "Comms Uplink" bottom quick-menu.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm installed on your local machine.

### Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/aceandro2812/ai-agent-portfolio-nexus.git
   ```
2. Navigate to the project directory:
   ```sh
   cd ai-agent-portfolio-nexus
   ```
3. Install NPM packages:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
